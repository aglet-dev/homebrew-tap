# homebrew-tap

Homebrew tap for [**Aglet**](https://aglet.dev).

```sh
brew install aglet-dev/tap/aglet      # CLI
```

The app (GUI) ships as a signed `.dmg` from <https://aglet.dev> with in-app
Sparkle auto-update — not via `brew --cask`.

`Formula/aglet.rb` is updated automatically by the release pipeline on each tag.
